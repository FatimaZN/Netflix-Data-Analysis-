# Netflix Data Analysis

## Overview
This project analyzes a dataset of Netflix movies and TV shows using Python.

## Objectives
- Explore the dataset
- Clean missing data
- Analyze trends in movies vs TV shows
- Analyze which countries produce the most content
- Visualize key insights

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
- There are more movies than TV shows on Netflix
- Most Netflix content comes from the United States
- Missing data was cleaned
- Basic visualizations created
